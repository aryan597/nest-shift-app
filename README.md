# NestShift

A Flutter app for controlling your NestShift smart home hub. Features QR pairing, device control, AI assistant, automations, and insights.

## Features

- **QR Pairing** - Scan hub QR code to connect
- **Hub Control** - Manage GPIO relays and sensors
- **AI Assistant** - Voice-controlled home commands
- **Automations** - Create and manage automation rules
- **Demo Mode** - Test without hardware

## Setup

```bash
flutter pub get
flutter run
```

## Tech Stack

Flutter, Riverpod, GoRouter, WebSockets, mobile_scanner
